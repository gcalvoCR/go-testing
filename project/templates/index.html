üí≥ Account Management

GET /accounts
Retrieve all accounts
Response:
[ { "id": "1", "name": "John Doe", "balance": 1500.50, "currency": "USD", "created_at": "2025-01-17T10:30:00Z", "updated_at": "2025-01-17T12:15:00Z" } ]
POST /accounts
Create a new account
Request:
{ "name": "Alice Johnson", "balance": 1000.00, "currency": "USD" }
Response:
{ "id": "2", "name": "Alice Johnson", "balance": 1000.00, "currency": "USD", "created_at": "2025-01-17T14:30:00Z", "updated_at": "2025-01-17T14:30:00Z" }
GET /accounts/{id}
Get account by ID
Response:
{ "id": "1", "name": "John Doe", "balance": 1500.50, "currency": "USD", "created_at": "2025-01-17T10:30:00Z", "updated_at": "2025-01-17T12:15:00Z" }

üí∏ Transaction Management

GET /accounts/{account_id}/transactions
Get all transactions for an account
Response:
[ { "id": "1", "account_id": "1", "amount": 500.00, "type": "deposit", "created_at": "2025-01-17T10:35:00Z", "updated_at": "2025-01-17T10:35:00Z" }, { "id": "2", "account_id": "1", "amount": 100.00, "type": "withdrawal", "created_at": "2025-01-17T11:00:00Z", "updated_at": "2025-01-17T11:00:00Z" } ]
POST /transactions
Create a new transaction (deposit/withdrawal)
Deposit Request:
{ "account_id": "1", "amount": 250.00, "type": "deposit" }
Withdrawal Request:
{ "account_id": "1", "amount": 100.00, "type": "withdrawal" }
Response:
{ "id": "3", "account_id": "1", "amount": 250.00, "type": "deposit", "created_at": "2025-01-17T15:45:00Z", "updated_at": "2025-01-17T15:45:00Z" }

üí± Exchange Rates
